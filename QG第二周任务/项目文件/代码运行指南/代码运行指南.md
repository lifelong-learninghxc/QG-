# 车辆智能编队控制复现项目  
本项目是对《Feedback-based platoon control for connected autonomous vehicles under different communication network topologies》论文核心实验的完整代码复现（任务A）。本指南将指导您如何成功运行代码，并详细说明代码运行后生成的所有结果文件。

## 1. 环境准备

在运行本代码之前，请确保您的计算机上已安装 Python 环境，并安装了以下两个必须的第三方依赖库：

* `numpy` (用于矩阵推导和核心动力学数值计算)
* `matplotlib` (用于数据可视化、三线表绘制和 GIF 动图生成)

**安装命令：**
```bash
pip install numpy matplotlib
```

---

## 2. 代码运行步骤

本代码设计了人性化的交互式控制台，您可以按照以下步骤轻松运行：

**第一步：启动程序**
在终端或您常用的 IDE 中运行主程序文件：
```bash
python 任务A复现代代码.py
```

**第二步：选择参数输入方式**
程序启动后，控制台会提示您选择数据的输入方式。为了快速验证论文的复现结果，**强烈建议您选择默认选项**：
* **输入 `1` 并回车（推荐）**：使用文献 Table 1 中预设的默认参数（包含 1 辆领航车和 3 辆跟随车的初始位置与速度），直接开始仿真。
* *备选项 `2`*：从同级目录下的 `config.json` 文件读取自定义配置。
* *备选项 `3`*：在控制台手动逐一输入各项初始状态参数。

**第三步：等待仿真完成**
程序将按顺序全自动执行以下三个核心任务，您只需等待控制台打印进度即可：
1.  计算并仿真 **Case I (全连通拓扑)** 的车辆运动状态。
2.  计算并仿真 **Case II (部分通信中断拓扑)** 下系统的鲁棒性表现。
3.  利用坐标系精确绘制并生成符合 IEEE 标准格式的 **Table 2 性能对比三线表**。

## 3. 代码生成结果说明

为避免文件混乱，代码运行之初会自动在您的计算机**桌面**创建一个名为 `2026QG_Platoon_Results` 的专用文件夹。所有的实验输出结果都会保存在这里。

运行结束后，打开该文件夹，您将看到以下 9 个文件：

### 📌 Case I：全连通拓扑仿真结果 (理想状态)
本组结果展示了通信通畅时，车辆如何完美汇聚成平行车队：
* **`Case_I_Position_Trajectories.png`**：二维平面内的车辆位置轨迹图，展现车辆从散乱分布到排成直线的运动全过程。
* **`Case_I_Gap_Trajectories.png`**：横向与纵向间距误差随时间的变化折线图，可观察误差最终平稳归零的过程。
* **`Case_I_Velocity_Trajectories.png`**：各车辆 X 轴与 Y 轴方向的速度变化折线图，展现所有跟随车速度收敛至领航车速度（6m/s）的过程。
* **`Case_I_platoon_animation.gif`**：直观的车辆编队运动动态可视化 GIF 图。

### 📌 Case II：部分通信中断仿真结果 (鲁棒性验证)
本组结果展示了当车辆 $i$ 掉队失去通信时，剩余车辆（$i+1$ 和 $i+2$）的重组表现：
* **`Case_II_Position_Trajectories.png`**：位置轨迹图（可观察到掉队车辆的独立轨迹与剩余车辆的稳定编队）。
* **`Case_II_Gap_Trajectories.png`**：重组车队的间距误差收敛过程。
* **`Case_II_velocity_Trajectories.png`**：重组车队的速度收敛过程。
* **`Case_II_platoon_animation.gif`**：展示部分断连后，系统动态重组为小编队的 GIF 动图。

### 📌 核心数据对比表
* **`Table_2_Performance_comparisons.png`**：这是由代码直接生成的图像文件。它完美复原了原论文中的 Table 2，直观对比了两种 Case 下系统在纵向/横向间距收敛时间、速度收敛时间以及振荡幅度上的性能表现差异。
